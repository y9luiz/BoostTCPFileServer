## Como compilar?

    mkdir build
    cmake ..
    make

    PS.: 
        necessário suporte para C++ 17
        Testado usando g++ 7.5
        CMake  3.20.0
        GNU Makefile  4.1
        Testado usando a boost ver 1.7.5

## Como rodar?

    Inicie o servidor (utilize sudo se necessário):

        ./launch_server 

    Inicie o cliente:

        ./launch_cliente

## Como configurar?

Altere o conteúdo do arquivo `config_server.txt`
